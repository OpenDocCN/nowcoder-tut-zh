**鸽巢定理**：n 个鸽子巢，若有 n+1 只鸽子在里面，则必有一个巢里至少有 2 只鸽子。
**鸽巢定理的推广**：设 k 和 n 都是任意的正整数，若至少有 kn+1 只鸽子分配在 n 个鸽巢里，则至少存在一个鸽巢中有不少于 k+1 只鸽子。
（1）4 项培训任选 2 项的组合数有 4*3/2=6，即 n=6，
（2）无论如何安排，都至少有 5 名党员参加的培训完全相同，即 k+1=5，k=4
（3）所以 kn+1=25
