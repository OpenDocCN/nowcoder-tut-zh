C++中&有另外一个含义，用来声明引用，&不是地址运算符，而是类型标识符的一部分。就像声明 char*指的是指向 char 的指针一样，int &指的是指向 int 的引用，变量和变量的引用指向相同的值和内存单元，引用相当于变量的别名。因此在参数传递时也只需要传递变量名即可，即 Func(a)。它的形式上看起来与值传递相同，但它传递参数时不复制副本，与指针一样可以修改主调函数的参数值，且使用起来更方便。
