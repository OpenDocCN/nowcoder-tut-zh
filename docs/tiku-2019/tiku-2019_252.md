1+（100-2）+（100-2）/2=148。先取出两个元素，比较 1 次，较大的为 max，较小的为 min:之后的（100-2）个元素，每个与 max 和 min 比较。如果需要比较的元素比 max 大，或者比 min 小，则只需要比较一次，共比较（100-2）次；如果需要比较的元素比 max 小，则需再次与 min 比较。出现这种情况的概率为 0.5，所以需比较（100-2）/2 次；
