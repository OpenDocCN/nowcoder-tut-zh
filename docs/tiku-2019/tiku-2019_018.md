使用 new 操作符来分配对象内存时会经历三个步骤：

第一步：调用对象的**析构函数**。

第二步：编译器调用 operator delete(或 operator delete[])函数释放内存空间。

**总之来说，new/delete 会调用对象的构造函数/析构函数以完成对象的构造/析构。而 malloc 则不会。**
